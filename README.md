# Viulux-V9-Driver-for-SteamVR
将Relativty驱动和OpenHMD缝合而成的Steam VR驱动。姑且可以正常使用。
可配合Relaivty的摄像头全身追踪。

水平有限，Bug在所难免。

bug:

1. 偶尔启动SteamVR屏幕显示全红，重启SteamVR即可。
2. 头部转动时偶而会抽搐。原因我不知道。

可用功能：

1. 头显显示。
2. 头显3自由度追踪。
3. nolo cv1手柄按键及震动。
